Print Definition Modifier

Use this class to display extra information as written in the title tag in print mode only. If this is placed on an icon the icon will be replaced by the text.
Press Ctrl+P to see this in action

23

50000 €

<p><i class="icon-speechbubble m-definition-before" title="Comments:"></i> 23</p>
<p class="m-definition-before" title="Total budget:">50000 &euro;</p>

Print HREF Modifier

Use this class to display the href of a link after the link text in print mode only.
Press Ctrl+P to see this in action

Google 
<a class="m-print-href" href="http://google.com">Google</a>

Print Utility Classes

These classes display or hide things depending on whether print or screen. These should only be used if no other method is useable such as the modifiers listed in this category. Don't forget to hide from screen-readers where appropriate.
